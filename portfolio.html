Markdown Forms

Markdown Forms is a lightweight format spec and renderer for building fillable forms in plain Markdown. Author human-readable *.form.md files using simple paper-form patterns (blanks, checkboxes, radios, selects, text areas) with optional inline metadata and validation, then compile to clean HTML and schema output.
